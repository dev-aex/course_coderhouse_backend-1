# Instruments E-commerce API
API for a musical instrument e-commerce as a final project for the Backend-1 course at CoderHouse.

# Getting Started

## Install dependencies

```
$ npm install
```

## Deploy the project

```
$ npm run dev
```

# Build with

- Node.js
- Express.js
- Handlebars
- Socket.io
- Nodemon
- Moment.js
- SweetAlert2
- Multer
